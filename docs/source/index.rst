Welcome to 3KIDNAS documentation!
===================================

**3KDIDNAS** (3D-Kinematic Data aNalysis Algorithim for Surveys) is designed to generate kinematic models from 3D data cubes.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
